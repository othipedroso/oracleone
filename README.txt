# Análise de Vendas das Lojas do Seu João

##Descrição do Projeto

Este projeto realiza uma análise de vendas de quatro lojas do Seu João utilizando Python e bibliotecas de Data Science como Pandas e Matplotlib.  
O objetivo é gerar insights para tomada de decisão, como identificar quais produtos são mais vendidos, calcular o faturamento por loja e analisar a satisfação dos clientes.

##Funcionalidades Desenvolvidas

- Carregamento de dados de vendas de quatro lojas diferentes
- Cálculo do faturamento total das lojas
- Cálculo do faturamento individual por loja
- Análise de vendas por categoria de produto
- Identificação dos produtos mais vendidos (Top 5) e menos vendidos (Bottom 5)
- Análise da quantidade de avaliações por nota
- Cálculo da média de avaliação geral das lojas
- Cálculo do frete médio por loja
- Geração de gráficos de barras para visualização do faturamento por loja

##Como Rodar o Projeto

1. Clone este repositório ou copie os arquivos do projeto.
2. Instale o Python 3 na sua máquina, se ainda não tiver instalado.
3. Instale as bibliotecas necessárias rodando:

bash
pip install pandas matplotlib

4. Execute o arquivo principal do projeto:

bash
python analise_lojas.py

##Resultados Obtidos

- Faturamento Total: calculado somando o faturamento das quatro lojas.
- Faturamento por Loja: exibido separadamente para Loja 1, Loja 2, Loja 3 e Loja 4.
- Vendas por Categoria: análise das categorias de produtos mais e menos vendidos.
- Produtos Mais Vendidos: top 5 produtos em quantidade de vendas.
- Produtos Menos Vendidos: bottom 5 produtos em quantidade de vendas.
- Quantidade de Avaliações: agrupadas por nota de avaliação (1 a 5 estrelas).
- Média Geral de Avaliação: valor médio das avaliações feitas pelos clientes.
- Frete Médio por Loja: média dos valores de frete cobrados em cada loja.
- Gráfico de Faturamento por Loja: gerado para melhor visualização dos resultados.
